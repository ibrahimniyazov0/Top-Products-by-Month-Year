# Top Products by Month & Year

This project demonstrates how to combine SQL aggregate functions with window functions, and visualize the results in Excel.

## Overview
- Applied SQL `SUM()` to calculate total sales per product by month and year.
- Used SQL window functions (`DENSE_RANK`, `ROW_NUMBER`) to rank products within each period.
- Exported results into Excel Pivot Tables.
- Visualized **Top 15 products** with Pivot Charts.
- Added interactive filters for **Months** and **Years**.

## Files
- `Top Products by Month & Year.sql` → SQL queries (SUM + DENSE_RANK)
- `Top Products by Month & Year.xlsx` → Excel workbook with Pivot Table & Chart
- `Top Products by Month & Year.png` → Visualization

## Skills Demonstrated
- SQL: Aggregates (`SUM`), Window Functions (`DENSE_RANK`)
- Excel: Pivot Tables, Charts, Filters
